Welcome to netallocation's documentation!
=========================================

This package provides ready to use functions for flow and cost allocation on `PyPSA <https://github.com/PyPSA/PyPSA>`_ networks. 


So far the common flow allocation methods 

* Average Participation 
* Marginal Participation
* Equivalent Bilateral Exchanges
* Zbus Flow Allocation 

are supported. The high dimensional data is stored in `xarray <http://xarray.pydata.org/en/stable/>`_ Datasets.
